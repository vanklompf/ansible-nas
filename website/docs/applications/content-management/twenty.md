## Usage

Set `twenty_enabled: true` in your `inventories/<your_inventory>/group_vars/nas.yml` file.

Twenty CMS web interface can be found at [http://ansible_nas_host_or_ip:3014](http://ansible_nas_host_or_ip:3014).
